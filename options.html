Moodle Docs Viewer Settings

About

This extension allows you to view Word documents (.docx) directly in your browser when browsing Moodle or other websites. Documents are rendered locally in your browser without needing to download them first.

File Size Limit

Maximum file size to preview (in MB). Larger files will prompt for download instead.

How It Works

When you click on a .docx link, this extension:

  1. Downloads the file using your browser's session (preserving authentication)
  2. Opens a new tab with the built-in document viewer
  3. Renders the document directly in your browser

This means you can view documents from authenticated sources like Moodle without having to download them to your computer first.
